Banana Pi M1

Στο προϊγούμενο άρθρο κάναμε μια εισαγωγή στο τι είναι το BusinessPi το οποίο μπορείτε να διαβάσετε στην ανάρτηση :

  • Τι είναι το BusinessPi

Σήμερα θα κάνουμε μια παρουσίαση του Hardware δοκιμών που χρησιμοποιήθηκαν για την υλοποίηση του BusinessPi. Για τις ανάγκες του έργου έγινε εκτεταμένη έρευνα αγοράς για να επιλέξουμε τον ιδανικό συνδυασμό hardware και software ώστε να πληρούν τις εξής προϋποθέσεις :

  • Ικανοποιητικές επιδόσεις για συνθήκες μιας μικρής επιχείρησης (έως 3-5 χρήστες)
  • Συνδεσιμότητα και επεκτασιμότητα του hardware
  • Βέλτιστο λόγο αξίας προς τιμή

Αφού αποκλείσαμε κάποια μοντέλα (λόγοι κόστους/διαθεσιμότητας/επιδόσεων) επιλέξαμε και συγκρίναμε τα board που εμφανίζονται στον παρακάτω πίνακα. (Σημείωση η τιμή κόστους είναι ενδεικτική)

Έτσι καταλήξαμε ότι η πρώτη έκδοση του έργου BusinessPi να γίνει διαθέσιμη για το Banana Pi M1.

Παρόλα αυτά η εγκατάσταση των λογισμικών που διαθέτει το BusinessPi μπορεί να γίνει αυτόματα με χρήση του ειδικού λογισμικού που αναπτύξαμε, ανεξάρτητα από το Board ή το λειτουργικό Linux που διαθέτει το Board σας. Η παρουσίαση του λογισμικού θα γίνει τις επόμενες μέρες.

Συνοπτική περιγραφή του Banana Pi M1

Το Banana Pi M1 είναι ένας single-board υπολογιστής που κατασκευάζεται στην Κίνα. Χρησιμοποιεί την Allwinner Α20 SoC, διαθέτει διπύρηνο επεξεργαστή, 1GB RAM και η σύνδεση με το δίκτυο γίνεται διαμέσου της ενσωματωμένης ethernet.

Το βασικό και επίσημο λειτουργικό σύστημα του BananaPi είναι το Bananian το οποίο είναι μια βελτιστοποιημένη έκδοση του λειτουργικού συστήματος Debian GNU/Linux για την πλατφόρμα του Banana Pi.

Παρόλα αυτά όμως ο χρήστης μπορεί αν εγκαταστήσει και άλλες διανομές Linux. Ως λειτουργικά συστήματα υποστηρίζει :

  • Android
  • Ubuntu
  • Debian
  • OpenSUSE
  • Fedora

Μια εκτενείς λίστα διαθέσιμων λειτουργικών συστημάτων συμβατών με το Banana Pi M1 μπορείτε να δείτε εδώ http://www.banana-pi.org/download.html

Παρελκόμενα

Επειδή τα board στην αγορά έρχονται ως έχουν (χωρίς δηλαδή περιφερικά), για τις ανάγκες του έργου χρησιμοποιήσαμε επίσης:

  • μια SD Card Class 10 (για τις ανάγκες του OS/Software Stack)
  • καλώδιο MicroUSB (για παροχή ρεύματος)
  • καλώδιο δικτύου (για σύνδεση στο υφιστάμενο δίκτυο/internet)

τα οποία φαίνονται στην παρακάτω εικόνα όπως είναι συνδεδεμένα και μπορείτε να διαβάσετε στην ανάρτηση : επί το έργον
